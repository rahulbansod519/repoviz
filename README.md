# repoviz

Generate an interactive architecture diagram and plain-English explanation for any local code repository.

## What it does

- **File tree view** — collapsible tree of every file in the repo
- **Import graph view** — force-directed graph of intra-repo import dependencies
- **AI explanation** — 2-3 sentence description of what the project does (via OpenAI gpt-4o)
- **Getting started guide** — step-by-step contributor guide generated by AI
- **Markdown export** — optional `.md` file with the explanation and guide

Supported languages for import analysis: Python, JavaScript/TypeScript, Go, Java, Ruby.

## Installation

```bash
pip install -e .
```

Requires Python 3.11+.

## Usage

```bash
# Diagram only (no API key needed)
repoviz /path/to/repo --no-ai

# Full report with AI explanation
export OPENAI_API_KEY=sk-...
repoviz /path/to/repo

# Custom output filename
repoviz /path/to/repo -o my-report.html

# Also write a Markdown export
repoviz /path/to/repo --md
```

The output is a self-contained HTML file. Open it in any browser — requires an internet connection to load D3.js from CDN.

## Options

| Flag | Description |
|------|-------------|
| `-o, --output` | Output HTML filename (default: `<repo-name>-report.html`) |
| `--md` | Also write a Markdown export (same name, `.md` extension) |
| `--no-ai` | Skip OpenAI, generate diagram only |

## Dependencies

- [`click`](https://click.palletsprojects.com/) — CLI
- [`openai`](https://github.com/openai/openai-python) — AI explanation (optional with `--no-ai`)
- [D3.js v7](https://d3js.org/) — loaded from CDN at render time
